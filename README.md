# Premium Spearfishing App UI

This is a code bundle for Premium Spearfishing App UI. The original project is available at https://www.figma.com/design/8xOEwmdAqhhMFCrSok8p9I/Premium-Spearfishing-App-UI.

## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

**Abrir la app:** en la terminal verás una línea como `Local: http://127.0.0.1:5173/`. Abre esa URL en el navegador para ver la pantalla de inicio. Si el puerto 5173 está ocupado, Vite usará 5174, 5175, etc.; usa la URL que indique la terminal.
